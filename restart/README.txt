start.py can help  to initialize a firebase. You need change the url (firebase url) and start_file first. 

The basic framework of firebase:
|data: will include all text or force data (videos and pictures will be stored in cloud storage)
|-|0: the first data
|-|1: the second data
|object: will include all rows collected in log sheet
|-|0: the first row of log sheet
|-|1: the second row of log sheet
|user: will include all users' info
|-|userid1: the first user's info
|-|userid2: the second user's info
|setting: anything that will help you to retreive quickly
|-|setting_name1: ...
|-|setting_name2: ...
